Tecnologías básicas del desarrollo web

En esta página listaremos de manera breve las tres tecnologías básicas de la web

Cuales son estas tecnologías?

HTML

Es un lenguaje de marcado donde marcamos que elementos existen dentro de una p+agina utilizando etiquetas

CSS

Es un lenguaje de estilos que nos permite elegir como se ven y donde se se ubican los elementos de nuestra pagina web

Javascript

Es el lenguaje de programacion en el cual escribimos la intereactividad de nuestro sitio web

Por que querriamos aprender estas tecnologías?

Aprender desarrollo web nos beneficiará de distintas maneras:

  • Podremos buscar trabajo en un rubro que no para de crecer.
  • Nos habilita a trabajar para empresas de todo el mundo o a ser independientes.
  • Es un conjunto de tecnolog+ias no muy complicadas que nos introducen al mundo de la programacion.
